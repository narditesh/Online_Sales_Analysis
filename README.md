#  Online_Sales_Analysis
Analyzed an online sales dataset using Excel to identify sales trends, product performance, and regional distribution. Built pivot tables and interactive charts to track units sold over time and compare categories and regions. Designed a clean dashboard and summarized key insights to support data-driven business decisions.

#  Tools
  Excel (Pivot Tables, Charts)

#  Key Insights

- Units sold fluctuate with noticeable peak days.
- High-value categories (especially Electronics) drive most revenue.
- North America is the strongest-performing region.

#  Conclusion
The analysis shows stable sales performance with clear demand patterns across time, product categories, and regions. High-value products contribute most of the revenue, while regional differences highlight where the business performs best. These insights can help guide inventory planning, regional focus, and sales strategy.
